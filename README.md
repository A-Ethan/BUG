### 反和谐

1. 打开_classic_文件夹，在其中找到WTF文件夹，用记事本打开Config.wtf文件，将上述命令输入进去保存即可。
2. SET overrideArchive "0"  反骨头和墓碑和谐（主要对亡灵身上多余的肉）
3. SET profanityFilter "0"  反语言屏蔽（主要针对卧槽我操牛B了之类的语言）
